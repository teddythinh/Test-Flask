# Test Flask

## Installation

Clone the repository

```bash
git clone https://github.com/teddythinh/Test-Flask.git
```

Create a project folder and a venv folder within

```bash
py -3 -m venv venv
```

Activate the virtual environment

```bash
venv\Scripts\activate
```

install Flask

```bash
pip install Flask
```

## Run the app

```bash
flask --app test run
```
